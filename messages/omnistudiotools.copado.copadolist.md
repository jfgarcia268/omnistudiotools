# summary

Create JSON to bulk update a Copado User Story from a list of Type.APIName text file.

# examples

- Create manifest from list file:

  <%= config.bin %> <%= command.id %> --manifest manifest.txt

- Create manifest with username:

  <%= config.bin %> <%= command.id %> --manifest manifest.txt --username User123

# flags.manifest.summary

List.txt file location.

# flags.username.summary

Username to use.
