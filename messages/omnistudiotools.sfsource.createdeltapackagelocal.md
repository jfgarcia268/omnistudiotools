# summary

Create a delta package based on a local git hash.

# examples

- Create delta from hash:
  <%= config.bin %> <%= command.id %> -h f2a6eee1b509c3edd33ab070148be48e41242846 -d force-app

# flags.sourcefolder.summary

Source folder to create delta package from.

# flags.hash.summary

Git hash value to diff against.
