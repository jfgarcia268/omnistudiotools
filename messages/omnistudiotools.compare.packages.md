# summary

Compare two Vlocity metadata folders for overlap. Gives error if overlap found.

# examples

- Compare two folders for overlap:

  <%= config.bin %> <%= command.id %> --folder1 vlocity1 --folder2 vlocity2

- Compare with short flags:

  <%= config.bin %> <%= command.id %> -s vlocity1 -t vlocity2

# flags.folder1.summary

Folder 1 to compare.

# flags.folder2.summary

Folder 2 to compare.
